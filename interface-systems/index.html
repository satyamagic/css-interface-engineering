System-first interface architecture

Principles

  • Performance budget

    Initial load under 50KB. Time to interactive under 2 seconds.

  • Information hierarchy

    Visual weight follows data importance. No decoration without purpose.

  • Failure awareness

    Interfaces degrade gracefully when systems fail.

  • Accessibility by default

    Semantic HTML. Keyboard navigation. Reduced motion support.
